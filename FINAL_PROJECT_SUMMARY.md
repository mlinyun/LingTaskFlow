## 🎯 项目概述

LingTaskFlow是一个现代化的全栈任务管理系统，采用前后端分离架构：
- **后端**: Django 5.2 + DRF + JWT认证 + SQLite数据库
- **前端**: Vue 3 + Quasar Framework + TypeScript + Vite
- **设计理念**: 蓝白科技感配色，现代化UI/UX设计
