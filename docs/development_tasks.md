   - 所有公共方法必须添加文档注释

   - 所有用户输入必须验证和清理
   - 实现 CSRF 保护和 XSS 防护
   - 敏感操作需要权限验证

3. **性能要求**:

   - API 响应时间 < 500ms (95%的请求)
   - 前端首屏加载时间 < 2s
   - 数据库查询优化(避免 N+1 查询)

4. **用户体验要求**:
   - 所有操作提供即时反馈
   - 支持响应式设计(移动端适配)
   - 提供离线缓存功能
